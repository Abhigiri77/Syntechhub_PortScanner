# TCP Port Scanner

## Description
A multithreaded TCP Port Scanner developed in Python for the Syntechhub Cyber Security Internship.

## Features
- Scan TCP ports on a host
- Supports custom port ranges
- Multi-threaded scanning
- Detects open and closed ports
- Saves scan results to a log file

## Technologies Used
- Python 3
- socket
- threading
- datetime

## How to Run

```bash
python port_scanner.py
```

Enter:
- Target IP or Domain
- Start Port
- End Port

The results are also saved in `scan_results.txt`.
